# javascript-movie-review

# FE 레벨1 영화 리뷰 미션

## 기본 UI

- [x] 영화 목록 api를 호출한다.
- [x] 지금 인기 있는 영화 목록 20개를 보여준다.
- [x] 제일 먼저 있는 영화를 배너로 띄운다.
- [x] 더보기 버튼을 누르면 20개의 영화 목록을 추가로 불러온다.
- [ ] 더보기 버튼을 무한 스크롤로 대체한다.
- [x] 불러올 영화 목록이 없으면 더보기 버튼이 사라진다.
- [x] 영화 데이터를 받아오기 전에 스켈레톤 UI를 먼저 띄운다.

## 검색 시 UI

- [x] 검색어를 입력하고 버튼을 마우스로 누르거나 엔터키를 누른다.
- [x] 검색한 영화 목록 api를 호출한다.
- [x] 해당 검색어에 대한 영화 목록 20개를 보여준다.
- [x] 검색어를 검색 제목에 띄운다.
- [x] 더보기 버튼을 누르면 20개의 검색과 관련된 영화 목록을 추가로 불러온다.
- [ ] 더보기 버튼을 무한 스크롤로 대체한다.
- [x] 불러올 영화 목록이 없으면 더보기 버튼이 사라진다.
- [x] 검색 결과가 없으면 `검색 결과가 없습니다.` 라는 문구를 띄운다.
- [x] 영화 데이터를 받아오기 전에 스켈레톤 UI를 먼저 띄운다.

## 모달 UI

- [x] 영화 항목을 선택하면 모달 UI를 띄운다.
- [ ] 선택한 영화의 상세 정보 api를 호출한다.
- [ ] 해당 영화에 대한 상세 정보를 보여준다.
- [ ] 별점을 누르면 해당 별점에 맞게 나의 점수가 저장된다.
- [x] x 를 누르면 모달이 닫힌다.
- [x] esc 를 누르면 모달이 닫힌다.

## 예외 처리

- [x] 기본 UI에서 영화 정보를 불러오지 못했다면 `영화 정보를 불러오지 못했습니다. 다시 시도해주세요.` 라는 문구를 띄운다.
- [x] 영화 이미지를 불러오지 못했다면 `No Image` 이미지를 띄운다.
- [x] 검색 시, 영화 정보를 불러오지 못했다면 `영화 검색 결과를 불러오지 못했습니다. 다시 시도해주세요.` 라는 문구를 띄운다.
- [ ] 영화 상세 정보를 불러올 시, 정보를 불러오지 못했다면 에러 UI를 출력한다.

## 공통

- [ ] pc, 태블릿, 휴대폰에서 사용가능하도록 반응형 UI를 만든다.

## API 명세

### Popular

| 항목     | 내용                                                                                          |
| -------- | --------------------------------------------------------------------------------------------- |
| Method   | GET                                                                                           |
| Endpoint | /3/movie/popular                                                                              |
| 설명     | 현재 인기 있는 영화 목록을 조회하는 API입니다. 메인 페이지나 인기 콘텐츠 영역에서 사용됩니다. |

| Query Parameter | 타입   | 설명        |
| --------------- | ------ | ----------- |
| language        | string | 언어        |
| page            | number | 페이지 번호 |

---

### Search

| 항목     | 내용                                                              |
| -------- | ----------------------------------------------------------------- |
| Method   | GET                                                               |
| Endpoint | /3/search/movie                                                   |
| 설명     | 사용자가 입력한 키워드를 기준으로 영화 정보를 검색하는 API입니다. |

| Query Parameter | 타입   | 설명          |
| --------------- | ------ | ------------- |
| query           | string | 검색할 키워드 |
| language        | string | 언어          |
| page            | number | 페이지 번호   |

### E2E 테스트 시나리오

1. 첫 진입 시점

- api 호출
- 베너
- 리스트 (스켈레톤)
- 더보기 버튼

2. 더보기 클릭

- 더보기를 클릭했을 때
- api 호출
- 리스트 (스켈레톤)

3. 검색(데이터 있을때)

- 키워드 입력
- 검색버튼 클릭했을 때
- 검색어로 api 호출
- 제목 키워드 포함
- 리스트 (스켈레톤)

4. 검색(데이터 없을때)

- 키워드 입력
- 검색버튼 클릭했을 때
- 검색어로 api 호출
- 제목 키워드 포함
- 비어있을 때 문구 출력

5. 검색(더보기 없어지기)

- 키워드 입력
- 검색버튼 클릭했을 때
- 검색어로 api 호출
- 마지막 데이터에서 더보기 버튼이 사라지는지 확인
